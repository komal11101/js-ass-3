You are given an integer N, and your task is to print all the integers starting from 11 till N (N inclusive).
Note: You have to complete printNumbers function. No need to take any input.
Input Format
The first and the only line of the input contains an integer N.
Output Format
Print all the numbers starting from 11 till N (including N).
Constraints
1≤N≤500
Example
Sample Input
4
Sample Output
1 2 3 4
